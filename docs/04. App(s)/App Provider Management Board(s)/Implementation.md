# Implementation

### #1 "Last Changed" Apps
Last changed apps is showing the 4 apps latest changed.
<br>
The function is only supported, if the company has more then minimum 6 apps. Otherwise this section does not exist.
<br>

```diff
! GET: /api/apps/provided
```
